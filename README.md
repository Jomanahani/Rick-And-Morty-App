# Rick and Morty App

This project is a React application that displays characters and episodes from the Rick and Morty series. It uses **React**, **React Query (TanStack Query)**, **React Router**, and **styled-components**.

The app allows users to:  
- Browse all characters  
- Search characters by name  
- View detailed information for each character  
- See the list of episodes the character appeared in  

## Live Demo

You can see the app live here: [Rick and Morty App on Netlify](https://rickand-mortyapp.netlify.app/)
